# 📊 Sales Forecasting Dashboard

This project is a data-driven dashboard built using **Streamlit, Pandas, Plotly, and Scikit-learn**.  
It analyzes Superstore sales data, provides key performance indicators (KPIs), interactive visualizations, and forecasts future sales trends.

---

## 🚀 Features
- **KPIs**: Total Sales, Profit, Quantity
- **Interactive Charts**: Monthly trends, Region-wise analysis
- **Forecasting**: Predicts future sales using Linear Regression
- **Web Deployment**: Easily deployable via Streamlit Cloud

---

## 🛠 Tech Stack
- **Python**: Pandas, NumPy, Scikit-learn  
- **Visualization**: Plotly, Seaborn  
- **Framework**: Streamlit
